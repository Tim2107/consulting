# AI Governance Documentation

This folder separates board-level decisions, source-backed research, operational policy, and shared evidence tracking.

## Start Here

- [Situational Awareness Summary](./situational-awareness-summary.md) is the repo-safe handoff for a CTO, reviewer, or fresh AI session.
- [CONTEXT.md](../CONTEXT.md) defines the shared terminology used across the dossier.

## Folders

- [board-briefing](./board-briefing/) contains decision-oriented material for the AI Governance and Steering Board.
- [research-dossier](./research-dossier/) contains legal and technical research with sourced claims.
- [operational-policy](./operational-policy/) contains draft controls, approval rules, and process language.
- [evidence-register](./evidence-register/) contains shared evidence, risk, decision, and assumption IDs referenced by the other outputs.

## Alignment Rule

Substantive claims in the board briefing and operational policy should trace back to the research dossier or evidence register. Claims should be marked as current fact, near-term development, or speculation.
