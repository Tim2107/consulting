# Approved AI Tool Registry

Status: first operational draft for board review. This registry does not itself approve Client Confidential Data processing; it translates the current research into provisional operating rules and required gates.

## Status Legend

- **Allowed**: may be used within the stated data class and controls.
- **Restricted**: may be used only after the listed controls, routing, ownership, and review gates are satisfied.
- **Needs Decision**: evidence exists, but the board has not yet decided whether this route is acceptable.
- **Prohibited**: not allowed unless the board explicitly approves a Red Line exception.
