# Evidence Register

Shared register for sourced claims, risks, decisions, and assumptions.

Entries should use stable IDs and identify whether the statement is a current fact, near-term development, or speculation.

A current fact must be true now and supported by authoritative evidence.

A near-term development must be expected within six months and supported by concrete evidence such as official timelines, provider announcements, regulator consultations, or published product documentation.

Speculation must be clearly marked and must not be the sole basis for a governance rule.
